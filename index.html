Level 1 — Get Hired
0 to 6 months · Target: Endpoint Admin / Modern Workplace Engineer
Target salary: $70,000–$95,000
Goal: Pass MD-102, build a home lab, and land your first cloud/endpoint admin role. You don't need to know everything — you need to know enough to get hired and learn the rest on the job.
Phase A — Foundations (Week 1–2)
Phase B — Core Intune Skills (Week 3–8)
Phase C — Exam + Job Hunt (Week 9–12)
⚡
Level 2 — Survive & Grow on the Job
Month 6–18 · Target: Azure Admin + PowerShell Automation
Target salary: $95,000–$120,000
Goal: You're hired. Now go deeper. Study AZ-104 while working. Learn to automate with PowerShell. Understand hybrid environments. Go from "junior" to "mid-level."
Phase A — Azure Core (Month 6–9)
Phase B — PowerShell Automation (Month 7–12)
Phase C — Hybrid & Networking (Month 10–18)
🛡️
Level 3 — Work in Any Corporate Confidently
Month 18–36 · Security + Identity + Architecture
Target salary: $120,000–$150,000
Goal: Walk into any Microsoft environment, understand how it's set up, fix what's broken, and design what's needed. You know security, identity, compliance, and automation deeply.
Phase A — Security & Identity (Month 18–24)
Phase B — Governance & Compliance (Month 24–30)
Phase C — Architecture Thinking (Month 30–36)
🚀
Level 4 — Hero Mode
Year 3+ · Cloud Architect / Senior Engineer / Consultant
Target salary: $150,000+
Goal: Design full cloud environments, lead migrations, mentor juniors, and consult for any organisation. Remote-eligible, highly paid, and can work for yourself.
Phase A — Architecture (Year 3)
Phase B — Soft Skills & Leadership (Year 3–4)

Salary Progression
What you can earn at each level
Global averages. Salaries vary by country, company size, remote vs on-site. MSPs pay more at junior level. Enterprise pays more senior. Remote roles can pay US-equivalent salaries anywhere in the world.
Your earning trajectory

How to maximise your salary
Stack certifications
Each cert adds $10–25k to your market value. MD-102 + AZ-104 together are far stronger than either alone.
Target MSPs first
Managed Service Providers hire earlier, pay well, and expose you to many environments fast.
Go remote
Cloud admin is one of the most remote-friendly IT roles. Remote opens global salary ranges wherever you live.
Add PowerShell
Admins who automate with PowerShell are much harder to find and command significantly higher rates.

Where to apply
LinkedIn Jobs
Best platform. Set job alerts for "Intune", "Modern Workplace", "Endpoint Admin". Connect with recruiters daily.
MSPs (top priority)
Managed Service Providers hire earlier, train you, and expose you to many clients. Search "MSP Intune engineer" on LinkedIn.
Indeed / Glassdoor
Good volume of entry-level IT cloud roles. Filter by "cloud", "Intune", "endpoint". Good for salary benchmarking too.
Microsoft Partner Companies
Companies with Microsoft Gold/Silver partner status actively need Intune/Azure admins. Search "Microsoft partner employer".
Your current employer
Ask if there's a cloud team internally. Getting promoted from desktop support to endpoint admin at same company is the fastest route.
Upwork / Toptal
Once you have MD-102, freelance Intune/M365 admin work is available. Great for extra income while job hunting.
